# Release Notes: v0.1.0

**Release Date:** 2024-03-17

## Highlights

Initial release with Office document metadata extraction for DOCX, XLSX, and PPTX files.

## Added

- `uniofficeutil` package for Office document operations
- `ReadFileMetadata()` for extracting metadata from DOCX files
- `ReadFileMetadataPresentation()` for extracting metadata from PPTX files
- `ExportMetadata` struct for unified metadata representation

## Documentation

- README.md with installation and usage instructions

## Installation

```bash
go get github.com/grokify/gounidoc@v0.1.0
```

## Usage

```go
import "github.com/grokify/gounidoc/uniofficeutil"

// Read metadata from a DOCX file
metadata, err := uniofficeutil.ReadFileMetadata("document.docx")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Title: %s\n", metadata.Title)
fmt.Printf("Author: %s\n", metadata.Author)
fmt.Printf("Created: %s\n", metadata.Created)
```
