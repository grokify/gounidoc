# Release Notes: v0.4.0

**Release Date:** 2025-05-20

## Highlights

- Unified CLI with 27 commands for PDF and Office document operations
- MCP server with 12 tools for AI agent integration via omniskill pattern
- PDF merge, split, and page extraction capabilities
- Batch conversion with glob patterns and progress reporting
- XLSX and PPTX info commands for spreadsheet and presentation analysis

## Added

### CLI Commands

Unified Cobra CLI with 27 commands:

| Category | Commands |
|----------|----------|
| Conversion | pdf2docx, docx2pdf, pdf2png |
| PDF Operations | merge, split, extract, pagecount |
| Batch | batch-pdf2docx, batch-docx2pdf |
| Document Info | metadata, xlsx-info, pptx-info |
| PDF Text/Images | pdf-text, pdf-images, pdf2images, images2pdf |
| PDF Manipulation | pdf-watermark, pdf-rotate, pdf-compress, pdf-metadata |
| PDF Security | pdf-encrypt, pdf-decrypt |
| Office | docx-text, xlsx2csv, docx-replace |
| Server | serve, version |

### MCP Server

MCP server with 12 tools using omniskill pattern for AI agent integration:

- `pdf2docx` - PDF to DOCX with table/image/layout options
- `docx2pdf` - DOCX to PDF conversion
- `pdf2png` - Render PDF page to PNG
- `metadata` - Read Office document metadata
- `pdf_merge` - Merge multiple PDFs
- `pdf_split` - Split PDF into pages
- `pdf_extract_pages` - Extract specific pages
- `pdf_page_count` - Get page count
- `batch_pdf2docx` - Batch PDF to DOCX
- `batch_docx2pdf` - Batch DOCX to PDF
- `xlsx_info` - Get XLSX workbook info
- `pptx_info` - Get PPTX presentation info

### PDF Operations

- `MergePDFFiles()` - Combine multiple PDFs into one
- `SplitPDFFile()` - Extract each page as a separate file
- `ExtractPDFPages()` - Extract specific page ranges
- `GetPDFPageCount()` - Return total pages

### Batch Operations

- Batch PDF to DOCX conversion with glob pattern support
- Batch DOCX to PDF conversion with glob pattern support
- Progress callback support using `mogo/fmt/progress`
- `ExpandGlobPatterns()` and `FilterByExtension()` helper functions
- `BatchConversionResult` type with success/failure tracking

### Document Info

- XLSX info command returning sheet names and count
- PPTX info command returning slide count

### Typed Methods

Type-safe methods on Skill struct:

- `ConvertPDFToDocx()`, `ConvertDocxToPDF()`, `ConvertPDFPageToPNG()`
- `MergePDFs()`, `SplitPDF()`, `ExtractPDFPages()`, `GetPDFPageCount()`
- `ReadMetadata()`, `GetXlsxInfo()`, `GetPptxInfo()`
- `BatchConvertPDFToDocxWithProgress()`, `BatchConvertDocxToPDFWithProgress()`

## Installation

```bash
go install github.com/grokify/gounidoc/cmd/gounidoc@v0.4.0
```

## Environment Setup

```bash
export UNIDOC_KEY="your-metered-api-key"
```

## CLI Usage

```bash
# Start MCP server (default)
gounidoc

# Convert PDF to DOCX
gounidoc pdf2docx -i input.pdf -o output.docx

# Merge PDFs
gounidoc merge -i file1.pdf -i file2.pdf -o merged.pdf

# Batch convert with progress bar
gounidoc batch-pdf2docx -i "*.pdf" -d ./output/

# Get XLSX info
gounidoc xlsx-info -f workbook.xlsx
```

## Claude Desktop Integration

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "gounidoc": {
      "command": "/path/to/gounidoc",
      "args": ["serve"],
      "env": {
        "UNIDOC_KEY": "your-metered-api-key"
      }
    }
  }
}
```

## Library Usage

```go
import (
    "github.com/grokify/gounidoc/unipdfutil"
    "github.com/grokify/gounidoc/uniofficeutil"
)

// Merge PDFs
err := unipdfutil.MergePDFFiles([]string{"file1.pdf", "file2.pdf"}, "merged.pdf")

// Split PDF
outputPaths, err := unipdfutil.SplitPDFFile("input.pdf", "./pages/")

// Get XLSX info
info, err := uniofficeutil.GetXlsxInfo("workbook.xlsx")
fmt.Printf("Sheets: %v\n", info.SheetNames)

// Batch convert with progress
opts := unipdfutil.DefaultConversionOptions()
result := unipdfutil.BatchConvertPDFToDocxWithProgress(inputPaths, outputDir, opts, progressCallback)
```
