# Release Notes: v0.2.0

**Release Date:** 2024-03-24

## Highlights

Add unipdfutil package for PDF operations including PDF to DOCX conversion.

## Added

- `unipdfutil` package moved from github.com/grokify/goimage with PDF to DOCX conversion

## Changed

- Reorganize `uniofficeutil` package structure

## Installation

```bash
go get github.com/grokify/gounidoc@v0.2.0
```

## Usage

```go
import "github.com/grokify/gounidoc/unipdfutil"

// Convert PDF to DOCX
err := unipdfutil.ConvertPDFFileToDocxFile("input.pdf", "output.docx")
if err != nil {
    log.Fatal(err)
}
```
