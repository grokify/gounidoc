# Release Notes: v0.3.0

**Release Date:** 2025-01-25

## Highlights

- License support for UniPDF with environment variable configuration
- New docx2pdf CLI command for DOCX to PDF conversion

## Added

- License support for UniPDF via `SetLicenseAPIKey()` and `SetMeteredKey()`
- `SetMeteredKeyEnv()` for loading metered key from `UNIDOC_LICENSE_API_KEY` environment variable
- `cmd/docx2pdf` CLI command for DOCX to PDF conversion

## Changed

- Rename `ConvertDOCXFileToPDFFile()` for consistency and clarity

## Fixed

- Fix G115 integer overflow conversion lint warning in unipdfutil

## Dependencies

- Update github.com/unidoc/unipdf/v3 to 3.65.0
- Update github.com/unidoc/unioffice to 1.38.0
- Update github.com/grokify/mogo to 0.65.6

## Installation

```bash
go get github.com/grokify/gounidoc@v0.3.0
```

## Environment Setup

Set your UniDoc metered API key:

```bash
export UNIDOC_KEY="your-metered-api-key"
```

## CLI Usage

```bash
# Install the docx2pdf CLI
go install github.com/grokify/gounidoc/cmd/docx2pdf@v0.3.0

# Convert DOCX to PDF
docx2pdf -i document.docx -o output.pdf
```

## Library Usage

```go
import (
    "github.com/grokify/gounidoc"
    "github.com/grokify/gounidoc/uniofficeutil"
)

// Initialize UniDoc license from environment
if err := gounidoc.SetMeteredKeyEnv(); err != nil {
    log.Fatal(err)
}

// Convert DOCX to PDF
err := uniofficeutil.ConvertDOCXFileToPDFFile("document.docx", "output.pdf")
if err != nil {
    log.Fatal(err)
}
```
