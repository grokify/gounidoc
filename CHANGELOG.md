## [v0.4.0] - 2025-05-20

### Highlights

- Unified CLI with 16 commands for PDF and Office document operations
- MCP server with 12 tools for AI agent integration via omniskill pattern
- PDF merge, split, and page extraction capabilities
- Batch conversion with glob patterns and progress reporting
- XLSX and PPTX info commands for spreadsheet and presentation analysis

### Added

- Unified Cobra CLI with 16 commands: pdf2docx, docx2pdf, docx-metadata, pptx-metadata, merge, split, extract, pagecount, batch-pdf2docx, batch-docx2pdf, xlsx-info, pptx-info, mcp, version, help, completion
- MCP server with 12 tools using omniskill pattern for AI agent integration
- PDF merge functionality via `MergePDFFiles()` combining multiple PDFs into one
- PDF split functionality via `SplitPDFFile()` extracting each page as a separate file
- PDF page extraction via `ExtractPDFPages()` for specific page ranges
- PDF page count via `GetPDFPageCount()` returning total pages
- Batch PDF to DOCX conversion with glob pattern support
- Batch DOCX to PDF conversion with glob pattern support
- Progress callback support for batch operations using `mogo/fmt/progress`
- XLSX info command returning sheet names and count
- PPTX info command returning slide count
- Typed methods on Skill struct for type safety: `ConvertPDFToDocx()`, `ConvertDocxToPDF()`, `MergePDFs()`, `SplitPDF()`, etc.
- `ExpandGlobPatterns()` and `FilterByExtension()` helper functions for batch operations
- `BatchConversionResult` type with success/failure tracking and error details

### Dependencies

- Add `github.com/grokify/mogo/fmt/progress` for progress bar support in batch operations
- Add `github.com/mark3labs/mcp-go` for MCP server implementation
- Add `github.com/spf13/cobra` for unified CLI

### Documentation

- Comprehensive README.md with all 16 CLI commands and 12 MCP tools documented
- MkDocs site with getting-started, CLI reference, MCP server, library reference, and examples pages
